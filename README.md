## 🔒 Secure Cloud Storage System

**Multi-user encrypted cloud storage platform with distributed nodes and role-based access control**

This project implements a production-ready secure cloud storage system featuring end-to-end encryption, multi-user support, and granular access control mechanisms.

### Key Features
- 🔐 **End-to-End Encryption** - Military-grade encryption for all stored files
- 👥 **Multi-User Support** - Manage multiple user accounts with distinct permissions
- 🔑 **Role-Based Access Control (RBAC)** - Fine-grained permission management
- 🌍 **Distributed Nodes** - Scalable architecture across multiple storage nodes
- 🛡️ **Secure Authentication** - JWT-based session management
- 📊 **Admin Dashboard** - Monitor storage, users, and system health
